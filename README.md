
# MortalityPredictor
*A data analysis and machine learning project for predicting age of death using regression models.*

## Overview
**MortalityPredictor** aims to analyze demographic and health-related data to predict a person’s **age of death**.  
This project demonstrates skills in **data preprocessing, EDA, feature engineering, and regression modeling**.

## Project Workflow
1. **Data Preparation** – Clean and preprocess the `life_expectancy_data.csv` dataset.  
2. **EDA** – Explore trends and correlations using visualization.  
3. **Model Building** – Train a Linear Regression model to predict age of death.  
4. **Evaluation** – Measure performance using metrics like MAE, MSE, and R².  

## Technologies Used
- Python 3.x  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn
